GuiaUI.js

Librería de ayuda visual interactiva creada con JavaScript puro

¿Qué problema resuelve?

Muchas páginas tienen botones o secciones que el usuario no entiende al primer vistazo. GuiaUI.js ayuda mostrando explicaciones visuales paso a paso.

Componentes incluidos

TourJS

Recorrido interactivo que resalta elementos y muestra instrucciones.

TooltipJS

Mensajes flotantes de ayuda rápida para botones, textos o tarjetas.

Ejemplo de formulario

Pasa el mouse sobre los campos para ver los tooltips.
