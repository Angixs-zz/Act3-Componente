GuiaExplorador.js

Libreria de ayuda visual interactiva creada con JavaScript puro

¿Qué problema resuelve?

En internet o en muchas aplicaciones web nos encontramos con botones o secciones que el usuario no entiende al primer mommento de consultar la pagina. GuiaExplorador.js resuelve este problema, pues ayuda mostrando explicaciones visuales paso a paso.

Componentes incluidos

GuiaExplorador.js incluye dos componentes fundamentales para una buena exploracion del usuario

TourJS

Es una herramienta que sirve de recorrido interactivo que muestra instrucciones.

TooltipJS

Es una herramienta que muestra mensajes de ayuda rápida para botones, textos o tarjetas.

Ejemplo de formulario

Pasa el mouse sobre los campos para ver los tooltips.
